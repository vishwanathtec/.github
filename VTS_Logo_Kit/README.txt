Vishwanath Tec Systems (VTS) Logo Kit

Files Included:
- vts-logo.svg: Scalable vector logo (transparent background)
- vts-logo.png: 512×512 main logo for GitHub, docs, or profile
- vts-logo-banner.png: 1024×512 version for website headers or covers
- vts-favicon.png: 64×64 favicon for web use
- vts-dark-bg.png: 512×512 version optimized for dark backgrounds

Color reference:
Primary: #2B2B2B
Background: Transparent (or #FFFFFF for print)
Font: Modern geometric sans-serif (Poppins/Inter)

Usage:
Use SVG for high-resolution prints or scaling. PNG for general digital use.
